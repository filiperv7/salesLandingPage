Conteúdo

Usando as tecnologias mais atuais do mercado, você vai aprender a desenvolver sites dinâmicos, interativos e com as melhores práticas.

Se torne expert em HTML e CSS no Front-end

Usando essas tecnologias para criar um front-end semântico, bonito, interativo e com acessibilidade. Habilidades muito escassas no mercado de trabalho.

Faça seus sites com um ótimo hankeamento no Google

Usar tags semânticas e recursos de acessibilidade faz com que seu site tenha uma ótima posição nas buscas do Google.

Construa a melhor experiência do usuário

A User Experience é muito importante para deixar seus sites mais confortáveis e dinâmicos na navegação. Aprenda essa e outra boas práticas do CSS.

Manipule elementos HTML com JavaScript

Usando o JavaScript, manipule a DOM. Adicione e remova elementos e animações, tornando seu site dinâmico e interativo.

Crie Back-end com Node e frameworks js

Com o Node você pode criar o back-end de sites usando JavaScript. Aprenda as melhores práticas do JavaScript no back-end.

Desenvolva Bancos de Dados

Use a linguaguem de consulta de banco de dados SQL para construir e manipular suas bases de dados no back-end.
